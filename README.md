# 📢 复读器

> 让机器人帮你发消息、复读、撤回，并支持自动撤回。

## 🚀 怎么用

**1. 发送文字**
```
/发送 你好呀
```
（也可用 `/发` 或 `/文本`）

--- 
**2. 复读消息**
- 引用一条消息，然后输入：
```
复读
```

<img src="./jpg/1.jpg" width="300">

---

**3. 撤回消息**
- 引用一条消息，然后输入：
```
撤回
```

#### 示例
<img src="./jpg/2.jpg" width="300">

---

**4. 自动复读机**
群内不同人连续发两条相同内容，机器人自动复读。

**5. 自动撤回**
机器人自己发的消息（通过 `/发送` 或 `复读`）会在设定秒数后自动消失。

## ⚙️ 配置（WebUI 可调，立即生效）

- **权限**：`发送`/`复读`/`撤回` 设为 `true` 仅管理员可用，`false` 所有人可用。
- **自动复读**：`复读机` 开关。
- **自动撤回**：`自动撤回` 开关 + `延迟时间`（秒，支持小数，默认 20）。

## ⚠️ 注意事项

- 自动撤回**只撤回本插件发出的消息**。
- 机器人需要**删除消息权限**才能撤回。
- 引用消息后输入命令才有效。
